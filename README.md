# Справочная система "Химия" (Вариант 1)

## 1. Описание проекта
Данный проект представляет собой разработку справочной системы для изучения тем школьного курса химии.
Система позволяет просматривать учебный материал, работать с интерактивной таблицей Менделеева и искать химические элементы.

Тип работы: Техническое задание №1

---

## 2. Цель и задачи проекта
**Цель:** разработка справочной системы для изучения химических элементов и основных тем курса химии.

**Задачи:**
- реализация навигации по учебным темам
- создание интерактивной таблицы Менделеева
- поиск элементов по названию, символу или атомному номеру
- валидация пользовательского ввода
- хранение данных в PostgreSQL

**Область применения:** образовательные учреждения, самообразование

---

## 3. Функциональность
- просмотр учебных тем
- поиск химических элементов
- отображение подробной информации об элементе
- интерактивная таблица Менделеева

**Ограничения:**
- требуется подключение к PostgreSQL
- работа только в среде Windows (WPF)

---

## 4. Архитектура проекта
**Тип архитектуры:** клиент-сервер (WPF + PostgreSQL)

**Технологии:**
- C# (.NET 8.0)
- WPF (Windows Presentation Foundation)
- PostgreSQL
- Npgsql
- xUnit

**Структура проекта:**
- `ChemistryApp/` — основной проект
- `ChemistryApp.Tests/` — проект тестов
- `Models/` — модели данных
- `ViewModels/` — модели представления (MVVM)
- `Views/` — представления (XAML)
- `Services/` — сервисы работы с БД
- `bin/Release/` — исполняемый файл

## 6. Использование
После запуска системы пользователь может:
- выбирать учебную тему из меню слева
- просматривать содержимое тем
- открыть таблицу Менделеева
- искать элементы по названию, символу или номеру
- кликнуть на элемент для просмотра подробной информации
